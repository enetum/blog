### Why use Bootstrap?

Not only is Bootstrap easy to use and speeds up the website development process, but it is also FREE! The code is open source and well documented as well. While it may seem daunting at first Bootstrap is easy to use and easy to learn. With Bootstrap you no longer have to write custom media queries for all your custom styles. You simply just need to add the classes you want to your HTML elements.
